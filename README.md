# Fila de Atendimento Hospitalar com Lista Encadeada

## 📌 Projeto Acadêmico

Projeto desenvolvido para a disciplina de Estrutura de Dados / Programação III.

Este sistema consiste na implementação de uma **fila de atendimento hospitalar com prioridade**, utilizando **lista encadeada simples**, onde pacientes com maior urgência são atendidos primeiro.

---

## 🧠 Descrição

O sistema simula um processo de triagem hospitalar, no qual os pacientes recebem um cartão indicando o nível de prioridade:

* 🟡 Amarelo (A): maior prioridade
* 🟢 Verde (V): menor prioridade

A estrutura de dados utilizada garante que:

* Pacientes com prioridade sejam atendidos primeiro
* A ordem de chegada seja respeitada dentro de cada nível de prioridade

---

## ⚙️ Funcionalidades

* Inserção de pacientes na fila
* Organização automática por prioridade
* Impressão da lista de espera
* Atendimento de pacientes (remoção do início da fila)
* Menu interativo no terminal

---

## 🛠️ Tecnologias Utilizadas

* Python
* Programação Orientada a Objetos
* Estruturas de Dados (Lista Encadeada)
