# hospital-fila-encadeada
Projeto desenvolvido para a disciplina de Estrutura de Dados / Programação III.

O sistema é dividido em duas partes principais:

Implementação de uma fila de atendimento hospitalar com prioridade, utilizando lista encadeada simples, onde pacientes com maior urgência são atendidos primeiro.
Implementação de uma tabela hash com encadeamento, representando estados brasileiros com base em uma função hash personalizada.

O objetivo do projeto é praticar conceitos fundamentais como:

Listas encadeadas
Filas com prioridade
Tabelas hash
Manipulação de ponteiros e estruturas de dados

Desenvolvido em Python para fins acadêmicos.
